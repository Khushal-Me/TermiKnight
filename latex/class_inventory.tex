Returns the inventory items as formatted strings. 

Generates a list of formatted strings representing the inventory contents.

Each item in the inventory is converted into a neatly formatted string. This is useful for aligned or side-\/by-\/side display.

\begin{DoxyReturn}{Returns}
A vector of strings, each representing a formatted item line.
\end{DoxyReturn}
The first line is a header ("{}=== Inventory ==="{}). If the inventory is empty, the next line will say "{}\+Empty"{}. Otherwise, each item is listed on its own line with a number and its name.

\begin{DoxyReturn}{Returns}
A vector of strings, each representing one line of formatted inventory output. 
\end{DoxyReturn}
