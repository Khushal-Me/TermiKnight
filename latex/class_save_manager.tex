\doxysubsection{Detailed Description}
Manages saving and loading game data. 

The \doxylink{class_save_manager}{Save\+Manager} class handles game state persitence by saving and loading player and world data to and from a save JSON file. It also includes a feature that will auto save and sabe before game termination.
