\doxysubsection{Detailed Description}
Represents the player\textquotesingle{}s character, storing health, attack, luck, inventory, and class-\/specific logic. 

The \doxylink{class_player}{Player} can equip weapons (increasing Attack), use potions, and activate class-\/specific skills during combat. The class also manages saving/loading of the player\textquotesingle{}s data via JSON.
