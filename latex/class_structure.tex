\doxysubsection{Detailed Description}
Represents a structure in the game that can contain rooms, enemies, and riddles. 

The \doxylink{class_structure}{Structure} class handles the exploration of rooms, managing riddles, spawning enemies, and rewarding the player with artifacts or items.
