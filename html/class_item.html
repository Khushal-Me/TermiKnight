Detailed Description

Represents a single in-game object that can be carried or used by the player.

Items can be consumables (like potions) or equipable (like weapons). Each Item has a name, a type, and an integer value that typically denotes how much it heals for (if a potion) or how much attack bonus it provides (if a weapon).
