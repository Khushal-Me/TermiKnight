Detailed Description

Manages turn-based combat sequences between a Player and one or more Enemies.

The CombatManager class provides static methods to initiate, run, and resolve combat interactions. It tracks special conditions (like blocking, class-specific skill usage) and determines when all enemies are defeated or when the player has fallen in battle.
